This project classifies Iris flower species using Machine Learning algorithms.

- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Models

- Logistic Regression
- Decision Tree
- Random Forest

## Evaluation
